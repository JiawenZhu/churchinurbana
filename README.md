# churchinurbana

A simple HTML/Node.js application with Express.

## Security Updates (January 2026)

This project has been updated to address security vulnerabilities identified by GitHub Dependabot.

### Packages Updated

| Package | Previous | Updated | Severity Fixed |
|---------|----------|---------|----------------|
| express | 4.17.1 | 4.22.1 | Multiple |
| qs | vulnerable | 6.14.1 | High |
| body-parser | vulnerable | 1.20.4 | High |
| cookie | vulnerable | 0.7.2 | Low |
| send | vulnerable | 0.19.2 | Low |
| serve-static | vulnerable | 1.16.3 | Low |
| path-to-regexp | vulnerable | 0.1.12 | High |

### Known Issues

> ⚠️ **swig is deprecated**: This project uses the `swig` templating engine which is no longer maintained and has known security vulnerabilities. Consider migrating to `ejs` or `nunjucks` for production use.

## Getting Started

```bash
npm install
npm start
```
